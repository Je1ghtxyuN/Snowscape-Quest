# Snowscape Quest

## Project Overview
Snowscape Quest is a VR therapy game designed for burn injury patients, providing a calming snowscape environment to aid pain management through distraction therapy.

## Current Status
The project is currently undergoing major game logic restructuring and continuous updates. Features and gameplay mechanics are subject to change during development.

## Technical Specifications
- **Engine**: Unity 2022 LTS
- **Render Pipeline**: Universal Render Pipeline (URP)
- **VR Framework**: XR Origin System
- **Platform**: SteamVR/OpenXR compatible devices
- **Development Status**: Active development

## Medical Application
Designed as a complementary therapy tool for burn injury rehabilitation. The game focuses on creating a peaceful, immersive experience to help patients manage pain through controlled VR interaction.

> **Medical Disclaimer**: This software is intended as a complementary therapy tool and should not replace professional medical treatment or advice.

# 雪境奇遇

## 项目概述
雪境奇遇是一款专为烧伤患者设计的VR治疗游戏，通过宁静的雪境环境，运用分心疗法帮助疼痛管理。

## 当前状态
项目目前正在进行大型游戏逻辑重构和持续更新。开发期间功能与游戏机制可能会有变动。

## 技术规格
- **引擎**: Unity 2022 LTS
- **渲染管线**: 通用渲染管线 (URP)
- **VR框架**: XR Origin 系统
- **平台**: 兼容 SteamVR/OpenXR 设备
- **开发状态**: 积极开发中

## 医疗应用
作为烧伤康复的辅助治疗工具设计。游戏专注于通过受控的VR交互，创造宁静的沉浸式体验来帮助患者管理疼痛。

> **医疗声明**: 本软件为辅助治疗工具，不能替代专业医疗治疗或建议。
